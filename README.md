# reCaptcha v3

Este es un ejemplo de implementación de Google reCaptcha v3

## Setup

Clona ambas paginas (index.html y procesar.php) a un servidor con php instalado.
No olvides setear tu clave pública de reCaptcha v3 en index.html, y tu clave privada en procesar.php.

Puedes seguir los pasos detallados en el siguiente tutorial <a target="_blank" href="https://walterdiaz.dev/tips/protegiendo-formularios-con-google-reCaptcha-v3">walterdiaz.dev/tips/protegiendo-formularios-con-google-reCaptcha-v3</a>, donde encontrarás una guía de como generar tus claves de reCaptcha v3 y una reseña de cómo funciona la codificación de index.html y procesar.php
